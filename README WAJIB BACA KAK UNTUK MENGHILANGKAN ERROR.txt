Jika ada error saat project dirun, silahkan delete dua file ini:
- ..\psd-lab-project\.vs\Project\config\applicationhost.config

- ...\psd-lab-project\Project\obj\Debug\Project.csproj.FileListAbsolute.txt

Setelah didelete, coba buka ulang projectnya dengan cara close Visual Studio dan open kembali project.sln nya

dan jika ada erorr lagi, coba untuk rebuild projectnya dengan cara:
1. ke tab solution explorer
2. klik kanan di solution, dan klik rebuild solution
3. klik kanan di project, dan klik rebuild project
